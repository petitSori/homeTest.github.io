Color (확장 클래스명은 컴포넌트 제작시 변경될 수 있음)

기본 컬러를 기반으로 사용 목적에 따른 컬러를 정의합니다.

Token

Palette Color를 기반으로 사용 목적에 따른 컬러를 정의합니다.
Semantic Color는 컬러 자체의 hex값이 아닌, 사용 목적과 적용하는 인터페이스에 따라 제공합니다.

Color (공통정의) - fill (카테고리) - primary (우선순위) - benefit (의미)
Color - contents - secondart - sale
Color - border - tertiary - soft
Color - border - quaternary - accent

SCSS

컴포넌트에 적용시 토큰 변수로 적용
단, 마크업 내에 사용될 단독 클래스명은 토큰명과 다를 수 있습니다.

** 마크업 내 하드코딩 클래스명 예시 ** - backgoundColor : bg-{"color-"제외 토큰명}
- color : text-{"color-"제외 토큰명}
- border : bd-{"color-"제외 토큰명}, bd-{"color-"제외 토큰명}-{굵기}
