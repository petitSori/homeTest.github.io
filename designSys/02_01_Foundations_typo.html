Typography

정리 된 Typography Family를 기반으로 사용에 연관하여 고유의 토큰명을 지정하여 사용합니다.

Token

CSS 기본 구조로 typo.{device}.{role}.{size}.{weight}의 정의로 진행합니다
맨 앞 [타입]의 경우 토큰명이 길어짐의 우려로 축약하여 사용합니다.

typo.desktop.header.xxl.bold
typo.desktop.title.xs.medium
typo.mobile.body.m.regular

Header

토큰명을 기준으로 반응형과 Desktop, Tablet, Mobile 등을 함께 대응 하는 구조입니다

Token Name Role Desktop Mobile
typo.{device}.header.xl.bold 페이지 단위의 타이틀에 사용합니다.
(ex.Visual영역/ Header1)		 64pt / 120% / -1 텍스트 56pt / 120% / -1 텍스트
typo.{device}.header.xl.medium 페이지 단위의 타이틀에 사용합니다.
(ex.Visual영역/ Header1)		 64pt / 120% / -1 텍스트 56pt / 120% / -1 텍스트
typo.{device}.header.l.bold 페이지 단위의 타이틀에 사용합니다.
(ex.페이지 상단영역/ Header2)		 56pt / 120% / -0.8 텍스트 48pt / 120% / -0.8 텍스트
typo.{device}.header.l.medium 페이지 단위의 타이틀에 사용합니다.
(ex.페이지 상단영역/ Header2)		 56pt / 120% / -0.8 텍스트 48pt / 120% / -0.8 텍스트
typo.{device}.header.m.bold 섹션 단위의 타이틀에 주로 사용합니다.
(ex.섹션 타이틀영역/ Header3)		 48 / 120% / -0.6 텍스트 40 / 120% / -0.6 텍스트
typo.{device}.header.m.medium 섹션 단위의 타이틀에 주로 사용합니다.
(ex.섹션 타이틀영역/ Header3)		 48 / 120% / -0.6 텍스트 40 / 120% / -0.6 텍스트
typo.{device}.header.s.bold 섹션 단위의 타이틀의 하위 위계로 주로 사용합니다.
(Header4)		 40 / 120% / -0.5 텍스트 32 / 120% / -0.5 텍스트
typo.{device}.header.s.medium 섹션 단위의 타이틀의 하위 위계로 주로 사용합니다.
(Header4)		 40 / 120% / -0.5 텍스트 32 / 120% / -0.5 텍스트
